
This project was developed using Visual Studio Community 2019.
It has been tested to run locally in a dev environment.

1. There are 3 projects: webApp and WebSvc both need to be running. webSvcTests only runs unit tests.
2. Localhost URLs are hardcoded and may need ports or protocols changed.
3. URLs are hardcoded at the top of files:
    - webApp\wwwroot\js\history.js
    - webApp\wwwroot\js\home.js
